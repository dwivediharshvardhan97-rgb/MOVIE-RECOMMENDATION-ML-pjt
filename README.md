This project is a Movie Recommendation System developed using Python and Machine Learning. It recommends similar movies based on user selection using content-based filtering techniques. The goal of this project is to help users discover movies that match their interests.



✨ Features
Search for a movie
Get similar movie recommendations
Content-Based Recommendation
User-friendly interface
Fast recommendation results


🛠️ Technologies Used
Python
Pandas
NumPy
Scikit-learn
Streamlit (if you used it)
Pickle



📂 Dataset
TMDB 5000 Movies Dataset
TMDB 5000 Credits Dataset


⚙️ How It Works
Load the movie dataset.
Preprocess and clean the data.
Extract important features (genres, keywords, cast, crew, overview).
Convert text into numerical vectors.
Calculate movie similarity using cosine similarity.
Recommend the top similar movies.
